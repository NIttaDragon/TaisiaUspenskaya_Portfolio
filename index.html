Проекты

1. Оно тебе надо

Первая работа на курсе Фронтенд-разработчик. Простой лендинг с использованием HTML и CSS. Основной задачей было повторить макет пиксель в пиксель, для этого использовался плагин для Chrome PixelPerfect.

2. Сложно сосредоточится

Вторая работа на курсе Фронтенд-разработчик. Сайт-статья с возможностью изменения цветовой схемы по нажатию кнопки, или по-умолчанию из настроек пользователя.

3. Закрывающий тег

Третья работа на курсе Фронтенд-разработчик. Сайт с небольшими постами с впечатлениями от прохождения курса. Этот проект посвящён работе с анимацией, фильтрами изображений и запоминанию состояния объектов.

4. Место

Четвёртая работа на курсе Фронтенд-разработчик. Небольшая социальная сеть, позволяющая менять имя, фотографию и описание профиля, выкладывать картинки с подписями на общий сервер группы. Целью проекта было научиться работать с сервером: отправлять на него данные и принимать от него ответы, а также знакомство с WebPack.

5. Web-ларёк

Пятая работа на курсе Фронтенд-разработчик. Одностраничный сайт интернет магазина. Реализовано добавление товара в корзину, удаление из неё, оформление заказа с отправкой данных пользователя и товара на сервер. Цель проекта построения архитектуры приложений MVC и реализация основного функционала.
